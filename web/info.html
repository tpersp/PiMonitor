Info and how to use it

This page explains the main URLs, which page to use for what, and how to connect from another device. The URL list below updates itself from the current PiMonitor configuration.

Main pages

  1. Live view: open the landing page to see the current feed quickly.
  2. Configuration: use the configuration page to change resolution, FPS, capture device, stream mode, Wi-Fi, recordings, and snapshots.
  3. Info: use this page when you need the direct stream URLs or want to remember how the app is supposed to be used.

How to connect from another device

  • For VLC on a PC, use the RTSP URL shown here when PiMonitor is in H264_RTSP mode.
  • For dual use, keep PiMonitor in H264_RTSP mode and enable HLS in Configuration. VLC will use RTSP while the web page uses HLS.
  • MJPEG mode is still useful for simple browser-only viewing and troubleshooting.
  • If HLS is enabled, open the HLS URL in a browser or media player that supports .m3u8 playlists.
  • If a stream is not working, first confirm the mode and ports on the configuration page.

Typical workflow

  1. Connect the HDMI capture card or webcam.
  2. Open Configuration and choose the device, resolution, and stream mode.
  3. Save the configuration and wait a few seconds for the stream service to restart.
  4. Open Live view to confirm the feed locally.
  5. Open the RTSP or HLS URL from another device if you want remote viewing.

Useful notes

  • MJPEG is usually best for quick testing and browser viewing.
  • RTSP is usually best for VLC and other dedicated video clients.
  • Some webcams work better with auto-detected format, while others may need a manual format selection from the configuration page.
  • Saved Wi-Fi networks can be added and switched from the configuration page.
